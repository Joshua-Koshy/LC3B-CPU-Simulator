# LC-3b Instruction-Level Simulator

This folder provides an instruction-level simulator for the LC-3b architecture.  
It directly executes machine-code files, modelling each instruction in turn and allowing tracing and inspection of the architectural state.

- Loads object files generated by the assembler (Lab1) and simulates each instruction one at a time. :contentReference[oaicite:3]{index=3}
